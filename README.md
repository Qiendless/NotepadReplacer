# NotepadReplacer

一键替换 Windows 系统记事本为任意编辑器。

通过 IFEO（映像劫持）将 `notepad.exe` 重定向到你选择的编辑器（Notepad++、VS Code、Sublime、Vim 等 27 款自动检测）。双击即用，零依赖，单文件 EXE。

```
启用替换 → 系统调用记事本时自动跳转到你的编辑器并正确打开文件
恢复默认 → 一键还原
```

## 特点

- 单文件 EXE，既是配置界面也是中转启动器
- 自动检测 27 款常见编辑器，支持手动浏览
- 编辑器历史持久化，换机器自动重置
- 管理员权限检测 + UAC 自动提权
- 64/32 位双视图 IFEO 写入

## 使用

双击运行 → 选编辑器 → 启用替换。卸载：点"恢复默认记事本"。
