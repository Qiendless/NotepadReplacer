# 记事本替换工具（NotepadReplacer）

把系统默认的记事本（Notepad）替换成任意文本编辑器



## 文件夹内容

| 文件 | 作用 |
|------|------|
| `NotepadReplacer.exe` | **主程序**（已编译，双击即用） |
| `NotepadReplacer.cs`  | C# 源码（界面 + 注册表操作 + 中转启动） |
| `app.ico`            | 程序图标（蓝紫渐变磁贴，已嵌入 exe） |
| `app.manifest`       | 程序清单（asInvoker，避免每次打开文件都弹 UAC） |
| `build.bat`          | 一键调用系统 `csc.exe` 重新编译 exe |
| `LICENSE`            | MIT 开源许可证全文 |
| `使用说明.md`        | 本文件 |



### 重新编译（可选）
把上述文件放同一文件夹，双击 `build.bat` 即可用系统自带编译器生成新版 `NotepadReplacer.exe`，
不需要 Visual Studio，也不需要 PowerShell（编译用的是 `C:\Windows\Microsoft.NET\…\csc.exe`，Win7+ 自带）。

## 说明

- 首次启用会弹出 UAC，点"是"即可。
- 自动检测 27 款常见编辑器（Notepad++、VS Code、Sublime、Vim、EmEditor 等），装在非默认位置也能识别。
- 编辑器历史保存在同目录 `NotepadReplacer.cfg`，换机器自动重置。
- 程序**既是界面也是中转器**，请放在稳定位置，删除后替换失效。

## 卸载

用本工具点"恢复默认记事本"即可，或手动删除注册表项：

```
HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\notepad.exe
```

中的 `Debugger` 值。

## 工作原理

通过 Windows 的**映像劫持（IFEO）**，为 `notepad.exe` 设置"调试器"指向本程序。
系统调用记事本时实际执行：

```
NotepadReplacer.exe notepad.exe <要打开的文件>
```

中转程序会跳过第 0 个参数（`notepad.exe` 自身），
只把**真正要打开的文件**交给你的编辑器，
从而保证"用记事本打开某文件"时能正确打开该文件。

注册表位置：
```
HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\notepad.exe
    Debugger = "…\NotepadReplacer.exe"
```
编辑器路径与历史记录保存在本程序同目录下的 `NotepadReplacer.cfg` 文件中（无需管理员即可读写），中转时直接读取。
配置文件内记录本机硬件机器码（Windows MachineGuid，不随计算机改名而变化），若检测到 exe 被拷贝到其他机器（机器码不匹配），将自动清空历史记录重新开始。
